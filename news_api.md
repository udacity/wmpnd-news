## 域名 https://test-miniprogram.com
## GET /api/news/list

### 入参

| 参数名 | 类型      | 描述                                                |
|:-------|:----------|:----------------------------------------------------|
| type   | String    | 新闻类型，值为gn、gj、cj、yl、js、ty和other其中之一 |
